Git Cheat Sheet

Config
git config - make changes system wide, global and local
Create
git init - creates an empty git repository in the current directory
git clone - cloning remote repository
Workflow
git add - adding file to staging area
git commit - move and merge the snapshot in the staging area to local repository
git reset - unstage a file
git restore - unstage a file or clean changes in working directory
git checkout - clean changes in working directory
Inspect & History
git status - show staged and unstaged files
git diff - show line by line diff
git log - list commit History
git show - show commit diff
git ls-files - list the files in repository
File Operations
git clean - remove untracked files
git rm - remove tracked files
git mv - rename/move tracked files
Branching
git branch - create/list/delete branches
git checkout - switch to another branch
git switch - switch to another branch
git stash - stack uncommitted changes to temporary storage
git rebase - applies the commits of this branch on the target branch
git cherry-pick - copy commit from one branch to another
git tag - set a custom name to specific commit
Remote
git remote - remote repository
git push - pushing local changes to remote
git fetch - getting remote changes to repo
