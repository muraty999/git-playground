Git Cheat Sheet

Config
git config - make changes system wide, global and local
Create
git init - creates an empty git repository in the current directory
Workflow
git add - adding file to staging area
git commit - move and merge the snapshot in the staging area to local repository
git reset - unstage a file
git restore - unstage a file or clean changes in working directory
git checkout - clean changes in working directory
Inspect & History
git status
git diff
git log
git show
git ls-files
File Operations
git clean
git rm
git mv
Branching
git branch
git checkout
git switch
git stash
